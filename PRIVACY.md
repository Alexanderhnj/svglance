# Privacy Policy

SVGlance is designed to work locally on your Mac.

## Data Collection

SVGlance does not collect analytics, telemetry, personal information, or usage data.

## File Access

SVGlance can only scan folders that you approve through macOS permission prompts. Approved folder access is stored locally using macOS security-scoped bookmarks.

SVGlance uses approved folder access to:

- find SVG files in approved folders
- apply visible Finder icons to SVG files
- watch approved folders for newly added or changed SVG files
- let the built-in viewer move between SVG files in the same approved folder

SVGlance does not upload SVG files or send file contents to any server.

## Network Access

SVGlance does not require network access for SVG rendering or Finder icon generation.

## Removing Access

You can remove approved folders from the SVGlance menu bar app. Use `Manage...` to remove individual folders or `Forget All Approved Folders` to reset all stored folder permissions.

## Contact

For privacy or security questions, open an issue on the project repository or use the security contact listed in [SECURITY.md](SECURITY.md).
