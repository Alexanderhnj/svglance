SVGlance

Make white and light SVGs visible on macOS with a checkerboard viewer, transparent PNG export, and Finder icon treatment.

SVGlance is a native macOS app for designers and developers who work with transparent SVG files. macOS makes white SVGs unreadable in preview; SVGlance gives them a clear transparency background so they can be inspected quickly.

Beta note: this is currently an unsigned preview build, so macOS may show a warning until Developer ID notarization is ready.
